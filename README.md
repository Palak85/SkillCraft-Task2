# Neon Stopwatch Web Application

A modern and interactive Stopwatch Web Application built using **HTML, CSS, and JavaScript**.
This project features a futuristic neon-glow UI, responsive design, smooth animations, and accurate stopwatch functionalities including lap tracking.

---

# 🚀 Project Overview

This stopwatch application was developed as part of a Web Development Internship task to practice:

* JavaScript timing functions
* DOM manipulation
* Event handling
* Responsive web design
* Interactive UI/UX development

The project provides a premium stopwatch experience with a visually attractive neon-themed interface.

# ⚙️ Functionalities

## ▶ Start Button

Starts the stopwatch timer.

## ⏸ Pause Button

Pauses the running timer without resetting time.

## 🔄 Reset Button

Resets the stopwatch back to:

```bash
00:00:00:00
```

## ⚑ Lap Button

Records lap times dynamically and displays them in a styled lap section.
